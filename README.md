# Luméra Beauty Store

Özgün Luméra kimliğiyle hazırlanmış modern kozmetik mağazası, etkileşimli alışveriş deneyimi ve yönetim paneli.

## Canlı yayın

- GitHub Pages: https://r0yc0ld.github.io/lumera-beauty-store/
- Tam uygulama önizlemesi: https://lumera-beauty-store.ongorulemeyengelecek.chatgpt.site  
  (Özel Sites yayını olduğu için ChatGPT oturumu isteyebilir.)

## Öne çıkanlar

- Mobil ve masaüstüne uyumlu vitrin, arama, kategori ve ürün kartları
- Ürün detayı, favoriler, sepet ve örnek ödeme akışı
- Cilt ritüeli testi, kampanya alanları, marka ve bülten bölümleri
- Sipariş, içerik ve mağaza ayarlarını kapsayan yönetim paneli
- Yönetim panelinden iyzico / PayTR / özel sağlayıcı seçimi
- 3D Secure, test/canlı ortam ve anahtar adlarının güvenli yapılandırılması
- D1 tabanlı mağaza ayarları API'si ve denetim günlüğü
- Özgün Luméra logosu, hero görseli ve sosyal paylaşım görseli

## Kaynak kod

Tam uygulama kaynak arşivi: [lumera-source.zip](./lumera-source.zip)

Yerel geliştirme:

1. Arşivi açın.
2. npm install komutunu çalıştırın.
3. .env.example dosyasını .env.local olarak kopyalayın.
4. npm run dev ile başlatın.

## Güvenlik ve ödeme notu

Gerçek kart tahsilatı için seçilen sanal POS sağlayıcısında aktif işyeri hesabı, canlı API anahtarları, webhook doğrulaması ve yasal/ticari sözleşmeler gerekir. Gizli anahtarlar depoya eklenmez; yalnızca sunucu ortam değişkenlerinden okunacak şekilde tasarlanmıştır.

## Teknoloji

React, TypeScript, vinext, Cloudflare D1, GitHub Pages ve OpenAI Sites.
